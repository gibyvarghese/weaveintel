What is geneWeave?

geneWeave is a full-stack chat and admin dashboard app built entirely from @weaveintel/* packages. It demonstrates all core and advanced capabilities in a real-world, extensible application.

Key Features

  • Chat engine with tool calling, policy, and audit
  • Admin dashboard for tools, skills, policies, workflows, triggers, and more
  • Scientific validation pipeline (multi-agent, evidence, verdicts)
  • Live-agent mesh orchestration and observability
  • DB-driven configuration and extensibility

Admin Dashboard

  • Manage tools, skills, policies, workflows, triggers, and capability packs
  • View audit logs, health snapshots, and approval requests
  • Configure prompt versions, experiments, and fragments
  • Monitor live-agent mesh status and contracts

Scientific Validation Pipeline

geneWeave includes a multi-agent scientific validation workflow, where hypotheses are decomposed, tested, and reviewed by specialist agents. Evidence and verdicts are tracked in an immutable ledger for auditability and reproducibility.

Extensibility

  • Add new tools, skills, and policies via the admin UI or DB
  • Integrate external systems via MCP, A2A, or custom packages
  • Extend workflows, triggers, and live-agent meshes for new domains

How to Run geneWeave

  1. Clone the repo and install dependencies
  2. Set your provider key(s): export OPENAI_API_KEY=sk-...
  3. Run ./scripts/start-geneweave.sh
  4. Open http://localhost:3500 and sign up
